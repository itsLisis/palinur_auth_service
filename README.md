# Installation Requirements

To install the modules required for the successful execution of this project, follow these steps:

1. **Create a virtual environment:**
```
python -m venv venv
```

2. **Activate the environment (PowerShell):**
```
.\venv\Scripts\Activate.ps1
```

3. **Install the required modules:**
```
pip install -r requirements.txt
```

4. **Deactivate the environment when you're done:**
```
deactivate
```

**Note:** Add the venv/ directory to .gitignore to avoid committing your virtual environment to the repository.
